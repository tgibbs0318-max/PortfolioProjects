# Data Analyst Survey Dashboard

## Project Overview
This Power BI project analyzes survey data collected from individuals working in or pursuing careers in data analytics and related fields.

The dashboard provides insights into:
- Average salaries
- Job titles
- Favorite programming languages
- Work/life balance
- Difficulty breaking into the data field
- Geographic distribution of respondents

## Tools & Skills Used
- Power BI
- Power Query
- DAX
- Data Cleaning
- Data Visualization
- Dashboard Design

## Dashboard Features
- Interactive filters and slicers
- KPI cards
- Salary comparisons
- Geographic analysis
- Career satisfaction metrics

## Key Insights
- Data Scientists reported the highest average salaries
- Python was the most popular programming language
- Many respondents found it moderately difficult to enter the data industry
- Work/life balance ratings varied significantly by role
