# Airbnb Market Analysis Dashboard

## Project Overview
This Tableau dashboard analyzes Airbnb listing data to identify pricing trends, occupancy patterns, and factors influencing host performance.

The project explores how variables such as location, room type, reviews, and availability impact listing prices and customer demand.

## Tools & Skills Used
- Tableau
- Data Visualization
- Data Cleaning
- Calculated Fields
- Geographic Mapping
- Dashboard Design

## Dashboard Features
- Interactive filters
- Geographic heat maps
- Price trend analysis
- Neighborhood comparisons
- Review score visualizations
- Occupancy insights

## Key Insights
- Entire homes had significantly higher average prices
- Listings with higher review scores tended to have increased occupancy
- Certain neighborhoods consistently outperformed others in revenue potential
- Seasonal pricing trends were visible across multiple markets
